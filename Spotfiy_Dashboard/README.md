🎵 Spotify Analytics Dashboard
About the Project


I created this Spotify Analytics Dashboard in Power BI to explore and analyze Spotify music data. The dashboard helps users understand artist performance, track popularity, album distribution, follower statistics, and music release trends through interactive visualizations.


The design is inspired by Spotify's dark theme, using green accents and modern dashboard elements to create a clean and engaging user experience.


Dashboard Highlights

Key Insights

  •	A small number of artists account for a large share of total followers.
  
  •	Popularity scores vary significantly across artists and albums.
  
  •	Most tracks in the dataset are non-explicit.
  
  •	Track releases have increased considerably in recent years.
